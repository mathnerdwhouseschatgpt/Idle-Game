# Idle Civilization Prototype

Idle Civilization is a browser-first idle game prototype covering 20 eras and 500 producers. The runtime is now split into a TypeScript UI, a TypeScript simulation worker, and a small shared game engine module so the main thread stays focused on rendering.

## Requirements
- Node.js 18+
- npm
- Python 3.9+ for data generation and CLI simulation
- Modern browser with ES modules and Web Workers
- Rust + Cargo only if you want to build the Tauri desktop app

## Browser UI
Install dependencies, build the TypeScript runtime, and start the local server:

```bash
npm install
npm start
```

Open `http://localhost:8000/`. The repository-root `index.html` redirects to `web/`.

### Runtime Shape
- `web/src/game-core.ts` contains the typed simulation engine and cost/production math.
- `web/src/sim-worker.ts` runs ticking, purchases, automation, saves, and snapshot preparation off the UI thread.
- `web/src/main.ts` owns DOM rendering and lazily creates building cards only for opened eras.
- `web/assets/*.js` is generated by `npm run compile` and is what the browser loads.

### Key Features
- Worker-driven real-time production loop
- Per-second resource deltas and scientific notation toggle
- Lazy era rendering for the 500-building dataset
- Buy x1/x10/x100/max buttons and per-building automation
- Live Kardashev tracker, top producer list, milestone progress bars, and unlock notifications
- Auto-save, local save/export/import, and offline progress simulation up to 8 hours

## Command-Line Simulator
```bash
python src/idle_cli.py
```

Commands:
- `status` - View current resources, production rates, era, and Kardashev index.
- `list` or `list <era>` - Inspect available buildings and costs.
- `tick [seconds]` - Advance time manually.
- `run <seconds> [step]` - Auto-tick for a duration.
- `buy <id|name> [quantity]` - Purchase buildings.
- `save [path]` / `load [path]` - Persist or restore progress.
- `help` - Show all commands.
- `quit` - Exit safely.

## Desktop Build With Tauri
The old Electron wrapper has been removed. The desktop target is now Tauri, which uses the OS webview instead of bundling Chromium.

Install Rust from `https://rustup.rs/`, then run:

```bash
npm install
npm run tauri:dev
npm run tauri:build
```

The current workspace has WebView2 and MSVC available, but Rust/Cargo are required before Tauri can build.

## Web Build For itch.io
```bash
npm run web:build
```

This regenerates data, compiles TypeScript, copies the static web runtime into `dist/web/`, and produces `dist/idle-civilization-web.zip`.

## GitHub Pages Deployment
```bash
npm run pages:build
```

The workflow publishes `dist/web/`. The packaged artifact excludes TypeScript source and includes only HTML, CSS, compiled JS assets, and generated data.
